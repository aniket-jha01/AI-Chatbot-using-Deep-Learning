# AI-Chatbot-using-Deep-Learning

An E-commerce type chatbot made with the help of PyTorch and Natural Languange Toolkit.
Implements basic NLP techniques , like bag of words, tokenisation, stemming 
Consists of a 3-layer feed forward Neural Net.
Chatbot GUI is implemented using tkinter.
Easily Modifiable intents , can be trained according to requirements.
